# S-mile Legal Pages

Repository ini berisi halaman legal untuk aplikasi S-mile, terdiri dari:

- `index.html`
- `privacy.html`
- `terms.html`

Halaman ini digunakan untuk:

- GitHub Pages
- Tautan Privacy Policy di Google Play Store
- Tautan Terms of Service di aplikasi

## Deployment

GitHub Pages diaktifkan dari branch utama (`main`) pada folder root.
